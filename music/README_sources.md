## More sources to mine (all commercial-friendly)

| Source | License model | Best for |
|---|---|---|
| **OpenGameArt.org** (what we used) | Filter by CC0 / CC-BY | Game music, synthwave |
| **Pixabay Music** — pixabay.com/music | Pixabay Content License (CC0-equivalent, **no attribution**) | Huge catalog, all genres |
| **YouTube Audio Library** — studio.youtube.com | Mix of CC0 and "royalty-free" (ads allowed) | Polished tracks |
| **Free Music Archive** — freemusicarchive.org | Filter by CC0 / CC-BY | Wider artistic range |
| **Incompetech** (Kevin MacLeod) | CC-BY 3.0 (credit required) | Many "game-ready" tracks |
| **ccMixter** — dig.ccmixter.org | CC-BY / CC-BY-SA | Remix-friendly |
| **Freesound.org** | Mix of CC0 / CC-BY | Samples/SFX, some tracks |
