# Resources

Game assets: textures, models, and sounds. These can be downloaded from online sources or created from scratch.

## Directory Structure

```
resources/
  textures/    Block textures, entity skins, UI sprites (PNG)
  models/      3D models for entities and objects (OBJ, glTF)
  sounds/      Sound effects and music (OGG, WAV)
```

## Sources

See `17_RESOURCE_GUIDE.md` in the project root for open-source asset sources and licensing info.

## Licensing

When adding assets, include license info (author, source URL, license type) either in a `LICENSE` file within the subdirectory or as a comment in the filename/commit message. Prefer CC0/CC-BY assets.
